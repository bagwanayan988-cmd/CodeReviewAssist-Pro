# 🚀 CodeReview Assist Pro AI

An AI-powered web application that helps developers review source code, analyze GitHub repositories, evaluate resume ATS scores, and generate professional PDF reports.

---

## ✨ Features

### 🤖 AI Code Review
- Review code using AI
- Supports multiple programming languages
- Stores review history
- Download PDF report

### 🐙 GitHub Repository Analyzer
- Analyze public GitHub repositories
- AI-generated repository review
- Repository history
- PDF export

### 📄 Resume ATS Analyzer
- Upload PDF resumes
- AI ATS score analysis
- Resume history
- PDF report generation

### 👤 User Authentication
- Register
- Login
- Logout
- Secure password hashing

### 📊 Dashboard
- Statistics
- Review counts
- GitHub counts
- ATS reports
- Charts

### 🔍 Global Search
- Search AI Reviews
- Search GitHub Reviews
- Search ATS Reports

### 📄 PDF Reports
- AI Review PDF
- GitHub PDF
- ATS PDF
